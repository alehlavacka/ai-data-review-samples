# AI Data Review & Annotation Samples

This repository contains simple examples of AI-generated text evaluation and annotation.

The goal is to demonstrate structured review, clarity improvement, and basic labeling logic.

---

## Example 1 – Weak AI Output Evaluation

**Original Text:**

"AI is very useful for businesses because it helps improve processes and makes companies more efficient."

**Identified Issues:**

- Vague language  
- Lack of specificity  
- No measurable impact  
- Generic phrasing  

**Improved Version:**

"AI helps businesses automate repetitive tasks, reduce operational costs, and improve decision-making through data analysis."

---

## Example 2 – Pattern Identification

**Observed Pattern in Multiple Outputs:**

- Repetitive generic introductions  
- Overuse of phrases like “in today’s world”  
- Lack of concrete examples  

**Conclusion:**

The model tends to generate safe but generic statements without contextual depth.

---

## Example 3 – Basic Annotation Sample

**Text:**

"AI tools can analyze customer behavior and improve marketing strategies."

**Assigned Tags:**

- Topic: Artificial Intelligence  
- Domain: Business / Marketing  
- Intent: Informational  
- Tone: Neutral  

**Reasoning:**

The text provides general information about AI usage in marketing without persuasive language.
